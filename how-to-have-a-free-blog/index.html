如何拥有一个免费的博客？

发表于 2020-04-01
  • 做最好的自己
首页 教程

学习技巧是什么？我认为每当解决一个问题之后都应当总结并记录下来，方便以后回顾。~~ 虽然我学习并不好。~~ 我发现，很多独立博客中都有优质的资料，给我新的发现。
“对任何渴望进步的人来说，写博客 / 文章 / 回答对自己的成长帮助都是巨大的。频率和数量不必过分强求，有话要说就写写，没事就歇歇，随意点。因为坚持写作的那一刻起，你就已经开始受益。” 这是某个博主说的。
那么，如何拥有一个博客呢？

# 1. 使用非独立博客

你可以选择在 CSDN、简书或者 cnblogs 上建立自己的博客，只要注册成功就能使用，无需域名、服务器，十分简便快捷。但这些平台有不少限制，比如个性化功能比较弱，而且还有广告的干扰。除此以外，文章的内容可能还有限制，写一下较为敏感的内容还有被和谐的风险。

# 2. 使用静态网页托管服务

一些代码托管平台提供静态网页托管服务，国外的有 GitHub、Gitlab、Netlify，国内的有 Coding、Gitee，这些都可以用来搭建个人博客。（Gitlab 就算了，在访问速度也够慢了）以 GitHub 为例，开启 GitHub Pages 后，你将会拥有一个 {username}.github.io 的域名，你也可以绑定自己的域名。若想知道如何拥有一个免费的顶级域名，请看我的下一篇文章。需要注意的是，国内访问 Gitee Pages 最快，但绑定自己的域名是要付费的。 因此，可以选择同时把博客部署在多个平台上来提高访问速度。

# 3. 使用免费空间

如今，越来越多的免费空间关闭了。现在我唯一能找到的只有挖站否站长 Qi 创立的其云否，提供免费空间供个人博主使用，有效期一年，到期后可以续费继续使用。为了避免滥用，其云否的免费空间需要用 30 积分兑换，其实签个几次到就能获得。其云否免费空间用的是美国 BGP 服务器，既然是免费的，速度稍微慢点也能接受。他们提供免费的二级域名，还有自动续签的 SSL 证书。如果你想拥有一个免费的顶级域名的话，我的下一篇文章会详细介绍。
