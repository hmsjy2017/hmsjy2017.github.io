如何注册免费域名？

# 如果想拥有一个顶级域名，可以注册 Freenom 的域名，后缀有五种， .tk 、 .ml 、 .ga 、 .cf 、 .gq 。很多人可能都以为此域名已经无法注册了，其实还是可以的，只是坑比较多，这里就详细讲解一下。

先准备一个梯子，尽量用美国的，因为之前 Freenom 遭国人滥用，已屏蔽中国 IP，还会验证填写的地区与 IP 是否一致。
首先，开启全局代理，浏览器访问 Freenom 官网。
输入要注册的域名点 检查可用性 ，注意⚠️搜索的时候一定要带上后缀，否则会点注册后会显示不可用。另外，域名前缀至少填写 4 位，否则要付费。默认选中注册 3 个月，请改为 12 个月（到期前可以免费续费）。
请使用谷歌邮箱或其他国外邮箱注册账号 （不能用 Outlook 邮箱），点击 Freenom 发给你的链接验证。一定要填写美国人信息（没有的话百度搜索美国人信息生成）。
如果提示 遇到技术问题 ，请修改个人信息再次尝试。注册完，进入 My Domains 查看已注册的域名。推荐把域名解析到 DNSPod 或 Cloudflare。
⚠️注意：切记不要滥用！！否则会封禁账号！！

另外，JS.ORG 为开发人员及其项目免费提供了子域名，可以作为博客的域名。比起 GitHub Pages 提供的 .github.io 二级域名， .js.org 更简短。Freenom 的免费域名是要续费的，相比，JS.ORG 的提供的子域名无需续费即可永久使用。
不过，想获得此域名也是有前提的：你的博客需要在 GitHub Pages 上，并且文章太少也不能申请。
如果想申请，请看这篇文章：组织在召唤：如何免费获取一个 js.org 的二级域名。（不是我写的）
